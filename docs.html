Nova Documentation

Nova is based on Bulma, an elegant and easy to use framework built using flexbox. It's modular, 100% responsive and open source.

For more information please check the official documentation: https://bulma.io/

Getting Started

This part of the doc will help you to quick start your project and will give you a basic idea about how Nova works.

How to include Nova to your project

In order to fully use nova compoenets you need to include the following files to your project:

  • 1. nova.css: this file can be found under nova/css/nova.css

  • 2. pe-icon-7-stroke.css: this file is needed in order to use the Stroke 7 icons library

  • 3. A reference to the Open Sans font

  • 4. jquery-3.3.1.min.js: this file can be found under vendor/js/jquery-3.3.1.min.js

  • 5. nova.js: this file can be found under nova/js/nova.js

  • 6. Copy the img folder and paste it so it's at the same level as your css and js folder

That's it, by doing this you will have access to all the nova components. If you wish to customize your nova.css please refer to the SASS section in the docs.

Some nova components such as Tables, dropzones, selects require some additional javascript. In the complex components section of the docs you can find more information about each component and which javscript is necesary in order to make it work.

SASS

Nova is built using SASS. This allows you to customize you nova.css file, meaning you can include only the components (modules) you are going to use and ommit the rest. In this section we will explain the folder structure of the SASS modules and also how to build your own customized version of nova.

BEM

Nova uses BEM as a name metodology. BEM stands fot Block Element Modifier.

Each element is name with the __ syntax

Each modifier is named with the --syntax
